# 🏠 Pune House Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting house prices in Pune using Machine Learning techniques. It uses real estate data such as area, number of bedrooms (BHK), bathrooms, and location to estimate property prices.

The goal is to build a reliable and accurate model that can help users make data-driven decisions while buying or selling properties.

---

## 🚀 Features
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Model Training using:
  - Linear Regression  
  - Random Forest  
- Model Evaluation (R² Score, MAE)  
- Real-time Price Prediction  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  
- Streamlit (for web app - optional)  

---

## 📊 Input Features
- Location  
- Total Area (Square Feet)  
- BHK (Bedrooms)  
- Number of Bathrooms  

---

## 🎯 Output
- Predicted House Price 💰  

---

## 📖 Extended Description
This project demonstrates an end-to-end Machine Learning workflow for solving a real-world problem in the real estate domain.

The process starts with data preprocessing, where missing values, duplicates, and outliers are handled. Exploratory Data Analysis (EDA) is performed to understand relationships between features and price trends.

Feature engineering techniques are applied to improve model performance. Multiple models such as Linear Regression and Random Forest are trained and evaluated using metrics like R² Score and Mean Absolute Error.

The project can also be extended into a web application using Streamlit, allowing users to input property details and get instant price predictions.
